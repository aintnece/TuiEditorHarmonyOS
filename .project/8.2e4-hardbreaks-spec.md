## 二、CommonMark 换行规则（要实现的行为）

- **行尾两个及以上空格** + 行结束 → **硬换行**（`<br />\n`）。
- **行尾一个反斜杠** + 行结束 → **硬换行**。
- 其它行结束（`\n`）→ **软换行**（渲染成 `\n`）。
- 换行前的**尾随空格剥掉**，换行后**下一行行首空格也跳过**（不进内容）。
- （本项目里行尾的 2 空格此前作为单字符 ` ` Text 节点已入 `out`；在 `\n` 分支统一弹出计数判定。）

## 三、改造后的三分支

### 3.1 转义分支（42–60 行）— 增加 `\` + `\n` → 硬换行

```typescript
// ── 转义字符 / 反斜杠硬换行 ──
if (text[i] === '\\' && i + 1 < len) {
  // 反斜杠 + 行结束 → 硬换行
  if (text[i + 1] === '\n') {
    const br: AstNode = new AstNode(AstNodeType.HardBreak);
    out.push(br);
    i += 2;
    while (i < len && text[i] === ' ') { i++; }   // 跳下一行行首空格
    continue;
  }
  const escaped: string = text[i + 1];
  if (isEscapable(escaped)) {
    const txt: AstNode = new AstNode(AstNodeType.Text);
    txt.text = escaped;
    out.push(txt);
    i += 2;
    continue;
  } else {
    const txt: AstNode = new AstNode(AstNodeType.Text);
    txt.text = '\\';
    out.push(txt);
    i++;
    continue;
  }
}
```

> `\` 是文本最后一字符（`i + 1 >= len`）时，外层 `if` 不成立 → 落普通文本分支 → 字面 `\`（CommonMark：块末尾的反斜杠是字面，正确）。

### 3.3 软换行分支（76–82 行）→ 重写为统一换行处理

```typescript
// ── 行结束：软/硬换行（剥换行前尾随空格 + 跳下一行行首空格）──
if (text[i] === '\n') {
  // 弹出已入 out 的尾随空格 Text 节点并计数
  let spaceCount: number = 0;
  while (out.length > 0) {
    const last: AstNode = out[out.length - 1];
    if (last.type === AstNodeType.Text && last.text === ' ') {
      out.pop();
      spaceCount++;
    } else {
      break;
    }
  }
  const br: AstNode = new AstNode(spaceCount >= 2 ? AstNodeType.HardBreak : AstNodeType.SoftBreak);
  out.push(br);
  i++;
  while (i < len && text[i] === ' ') { i++; }   // 跳下一行行首空格
  continue;
}
```

> **递归安全**：commonmark 对每个行尾都做「剥尾随空格 + 判定软/硬 + 跳行首空格」，链接/删除线内文递归同理，不会误伤。
