## 背景：当前实现（要替换的代码，Blocks.ts）

```ts
/** HTML 块起始检测 */
function isHtmlBlockStart(line: string): boolean {
  const trimmed: string = line.trimStart();
  if (!trimmed.startsWith('<')) return false;
  const tags: string[] = ['<address', ... '<pre', '<script', '<style'];  // ❌ 把 pre/script/style 混进 type6
  const lower: string = trimmed.toLowerCase();
  for (let i = 0; i < tags.length; i++) { if (lower.startsWith(tags[i])) return true; }
  if (trimmed.startsWith('</')) return true;
  if (trimmed.startsWith('<!--') || trimmed.startsWith('<![CDATA[')) return true;
  if (trimmed.startsWith('<?')) return true;
  return false;
}

/** 解析 HTML 块 */
function parseHtmlBlock(state: ParseState): AstNode {
  const node: AstNode = new AstNode(AstNodeType.HtmlBlock);
  let html: string = '';
  while (!state.isEnd()) {
    const line: string = state.currentLine();
    html += line + '\n';
    state.nextLine();
    if (line === '') break;          // ❌ 所有类型都遇空行结束（错）
  }
  const textNode: AstNode = new AstNode(AstNodeType.Text);
  textNode.text = html.trim();        // ❌ 全块 trim 破坏缩进
  node.appendChild(textNode);
  return node;
}
```

**当前 bug**：① 不区分 7 种类型，所有类型一律「遇空行结束」——类型 1-5 应按各自的结束标记结束（且**包含中间空行**直到标记或 EOF）；② 类型 6 列表错误混入 `pre/script/style`（它们是类型 1，结束条件完全不同）；③ `html.trim()` 破坏块内缩进与首尾空行语义；④ 类型 7 不应打断段落（当前会打断）。

## CommonMark 4.6 规则（commonmark.js 0.31.2 权威实现，逐条照做）

HTML 块起始：行（最多 3 个前导空格；前导缩进 ≥4 列则不是 HTML 块，归缩进代码）的首个非空白字符是 `<`，且匹配下列**七型之一**。检测**按 1→7 顺序，首个匹配者胜**（顺序重要：type 1 在 type 6 之前，故 `<pre`/`<script`/`<style` 归 type 1）。

设 `s` = 行去掉「最多 3 个前导空格」后、**从首个非空白字符开始**的子串（用于类型匹配）；但**块内容存储用整行原文**（含前导空格，不 trim）。

### 七型起始条件（reHtmlBlockOpen，权威正则 → 逐条翻成 char-loop）
| 型 | 起始（`s` 满足） | 大小写 |
|----|------|------|
| 1 | `<` + 名字∈{`script`,`pre`,`textarea`,`style`} + **紧跟** (空白 \| `>` \| 行尾)。**注意终止符不含 `/`** | 名字不区分大小写 |
| 2 | `s` 以 `<!--` 开头 | — |
| 3 | `s` 以 `<?` 开头 | — |
| 4 | `s` 以 `<!` 开头且第 3 个字符是 ASCII 字母（A-Za-z） | — |
| 5 | `s` 以 `<![CDATA[` 开头（精确这 9 个字符） | — |
| 6 | `<` 或 `</` + 名字∈下方**Type6 名单** + **紧跟** (空白 \| `>` \| `/` \| 行尾) | 名字不区分大小写 |
| 7 | `s` 是一个**完整的开标签或闭标签**，其后**只剩空白直到行尾**（用 Inlines 的 `parseHtmlOpenTag`/`parseHtmlCloseTag` 判定，见下） | — |

**Type6 名单（精确，全小写比较）**：
```
address article aside base basefont blockquote body caption center col colgroup dd details
dialog dir div dl dt fieldset figcaption figure footer form frame frameset
h1 h2 h3 h4 h5 h6 head header hr html iframe legend li link main menu menuitem
nav noframes ol optgroup option p param section search summary table tbody td
tfoot th thead title tr track ul
```
（注意：**含 `search`，不含 `source`**；`pre`/`script`/`style`/`textarea` **不在** type6 名单——它们是 type1。）

**名字提取与终止符判定（type1 / type6 通用做法，避免 `p` vs `param` 前缀歧义）**：
- 从 `<`（type6 还要看是否 `</`）后开始，取连续的「字母/数字」run 作为 tagname（`[A-Za-z][A-Za-z0-9]*`，下划线/连字符不取——commonmark 的块级 type1/type6 名字仅字母数字即可覆盖名单），转小写后**精确匹配**名单（exact，非前缀）。
- 再看 tagname run **之后那一个字符**是否是该型的合法终止符：
  - type1 终止符 = 空白(空格/制表/换行—行内只有空格/tab，或行尾 EOL) **或** `>`。**不含 `/`**。
  - type6 终止符 = 空白 **或** `>` **或** `/` **或** 行尾。

### Type 7 判定（复用 Inlines，不重写语法）
- 仅当 type1-6 都不匹配时才测 type7（顺序保证）。
- `import { parseHtmlOpenTag, parseHtmlCloseTag } from './Inlines';`（先给它们加 export）。
- 设 `s` 首字符是 `<`。若 `s[1] === '/'` → `e = parseHtmlCloseTag(s, 0)`；否则 `e = parseHtmlOpenTag(s, 0)`。这两个函数返回**标签结束后的索引**，失败返回 **-1**。
- 若 `e >= 0` 且 `s.substring(e)` 全是空白（空格/制表，或为空）→ type7 命中。否则不是 HTML 块。
- ⚠️ type7 **不能打断段落**（见下「打断段落」一节）。

### 七型结束条件（reHtmlBlockClose）
- **类型 1-5**：把当前行**先收进块**，再判断该行**是否包含**对应结束标记（`indexOf >= 0`，非锚定，**首行也要判**）；包含则块在该行**结束**（含该行）。**中间空行也属于块内容**，不结束。若直到 EOF 都没遇到结束标记，则块延伸到 EOF（消费所有剩余行）。
  | 型 | 结束（行内**包含**） |
  |----|------|
  | 1 | `</script>` 或 `</pre>` 或 `</textarea>` 或 `</style>`（**不区分大小写** → 对行 `toLowerCase()` 后 indexOf） |
  | 2 | `-->` |
  | 3 | `?>` |
  | 4 | `>` |
  | 5 | `]]>` |
- **类型 6 / 7**：遇到**空行**（全空白或空字符串）结束，**空行本身不计入块内容**（不收进、不消费——留给外层 parseBlocks 处理）。直到 EOF 也结束。

## 目标实现（Blocks.ts，逐段可编译 ArkTS）

新增/重写以下函数（删掉旧的 `isHtmlBlockStart`、`parseHtmlBlock`）：

```ts
/** HTML 块类型检测：返回 0(非) 或 1-7。s 用「去前导(<4列)空白后」的子串匹配。 */
function htmlBlockStartType(line: string): number {
  // 跳过最多 3 列前导空白；≥4 列 → 非 HTML 块（缩进代码）
  let i: number = 0;
  let col: number = 0;
  while (i < line.length && col < 4) {
    const ch: string = line[i];
    if (ch === ' ') { col += 1; i += 1; }
    else if (ch === '\t') { col += 4 - (col % 4); i += 1; }
    else break;
  }
  if (col >= 4) return 0;
  const s: string = line.substring(i);
  if (s.length === 0 || s[0] !== '<') return 0;

  // type 1: < (script|pre|textarea|style) 终止符(空白|>|EOL，不含/)
  const t1: number = matchHtmlTagNameType(s, false);   // 见 helper
  if (t1 === 1) return 1;
  // type 2: <!--
  if (s.length >= 4 && s[1] === '!' && s[2] === '-' && s[3] === '-') return 2;
  // type 3: <?
  if (s.length >= 2 && s[1] === '?') return 3;
  // type 5: <![CDATA[  （先于 type4 测，因为 '[' 不是字母 type4 不会误吞，但顺序按 4 在前也安全；这里显式精确匹配）
  if (s.length >= 9 && s.substring(0, 9) === '<![CDATA[') return 5;
  // type 4: <! + ASCII 字母
  if (s.length >= 3 && s[1] === '!' && isAsciiLetterCh(s.charCodeAt(2))) return 4;
  // type 6: </? 名字∈名单 终止符(空白|>|/|EOL)
  const t6: number = matchHtmlTagNameType(s, true);
  if (t6 === 6) return 6;
  // type 7: 完整开/闭标签 + 仅空白到行尾
  let e: number = -1;
  if (s.length >= 2 && s[1] === '/') { e = parseHtmlCloseTag(s, 0); }
  else { e = parseHtmlOpenTag(s, 0); }
  if (e >= 0 && isAllWhitespaceFrom(s, e)) return 7;
  return 0;
}
```

> ⚠️ 顺序细节：上面把 type5（`<![CDATA[`）放在 type4（`<!`+字母）之前显式判，避免任何歧义；commonmark 原序是 2,3,4,5，但 `<![` 的 `[` 非字母 → type4 不会误命中 `<![CDATA[`，两种顺序结果相同。保持上面写法即可。

**`matchHtmlTagNameType(s, isType6)`**（统一提取 tagname + 判终止符）：
```ts
/** 提取 < 或 </ 后的字母数字 tagname，匹配 type1 名单或 type6 名单。
 *  isType6=false → 查 type1 名单(script/pre/textarea/style)，终止符不含 '/'；
 *  isType6=true  → 查 type6 名单，终止符含 '/'。
 *  命中返回 1 或 6，否则 0。 */
function matchHtmlTagNameType(s: string, isType6: boolean): number {
  let p: number = 1;
  if (isType6 && s.length > 1 && s[1] === '/') { p = 2; }   // type6 允许 </
  // type1 不允许 </（开标签语义），p 保持 1
  if (p >= s.length || !isAsciiLetterCh(s.charCodeAt(p))) return 0;
  const nameStart: number = p;
  p += 1;
  while (p < s.length && isAsciiAlnumCh(s.charCodeAt(p))) { p += 1; }
  const name: string = s.substring(nameStart, p).toLowerCase();
  const term: string = p < s.length ? s[p] : '';   // 终止符（行尾 → ''）
  const isWs: boolean = term === '' || term === ' ' || term === '\t';
  if (!isType6) {
    if (name === 'script' || name === 'pre' || name === 'textarea' || name === 'style') {
      if (isWs || term === '>') return 1;            // type1 终止符：空白|>|EOL（不含 /）
    }
    return 0;
  } else {
    if (isType6Name(name)) {
      if (isWs || term === '>' || term === '/') return 6;  // type6 终止符：空白|>|/|EOL
    }
    return 0;
  }
}
```

**结束标记判定**：
```ts
function htmlBlockCloseMatch(line: string, blockType: number): boolean {
  if (blockType === 1) {
    const lc: string = line.toLowerCase();
    return lc.indexOf('</script>')>=0 || lc.indexOf('</pre>')>=0 || lc.indexOf('</textarea>')>=0 || lc.indexOf('</style>')>=0;
  }
  if (blockType === 2) return line.indexOf('-->') >= 0;
  if (blockType === 3) return line.indexOf('?>') >= 0;
  if (blockType === 4) return line.indexOf('>') >= 0;
  if (blockType === 5) return line.indexOf(']]>') >= 0;
  return false;
}
```

**重写 `parseHtmlBlock`**：
```ts
function parseHtmlBlock(state: ParseState): AstNode {
  const node: AstNode = new AstNode(AstNodeType.HtmlBlock);
  const firstLine: string = state.currentLine();
  const blockType: number = htmlBlockStartType(firstLine);   // 调用方已保证 >0
  const lines: string[] = [];
  while (!state.isEnd()) {
    const line: string = state.currentLine();
    if (blockType >= 6) {
      // type 6 / 7：空行结束（空行不计入、不消费）
      if (isBlankLine(line)) break;
      lines.push(line);
      state.nextLine();
    } else {
      // type 1-5：先收进，再判结束标记（首行也判）；中间空行也收
      lines.push(line);
      state.nextLine();
      if (htmlBlockCloseMatch(line, blockType)) break;
    }
  }
  const textNode: AstNode = new AstNode(AstNodeType.Text);
  textNode.text = lines.join('\n');     // ⚠️ 不 trim！renderer 负责补尾随 \n
  node.appendChild(textNode);
  return node;
}
```

```ts
/** 任意 HTML 块起始（type 1-7）——用于 tryParseBlock 的新块入口 */
function isHtmlBlockStart(line: string): boolean { return htmlBlockStartType(line) > 0; }

/** 可打断段落/懒续行的 HTML 块（type 1-6；type 7 不能打断段落） */
```

## ArkTS 硬约束（harness 绿 ≠ 能编进鸿蒙；必须遵守）
- **禁匿名对象字面量**（`return {a:..}`）；本批返回都是 `number`/`boolean`/`string`，无此风险。
- **禁 `Record`/索引签名**：名单用 `string[]` + `indexOf`，**不要**用对象/Record。
- **禁 `obj[key]` 对象属性下标**；`s[i]` 字符串取字符、`arr[i]` 数组下标、`s.charCodeAt(i)` **合法**。
- **禁正则**（`new RegExp`/`.replace(/../)`）：全部 char-loop / `indexOf` / `substring` / `toLowerCase`。
- **import 在文件顶部**：`parseHtmlOpenTag`/`parseHtmlCloseTag` 的 import 加到 Blocks.ts 顶部既有 import 区。
- 标准方法 `String.toLowerCase()` / `substring` / `indexOf` / `Array.indexOf` / `Array.join` 可用。
